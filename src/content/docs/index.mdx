## What is FeltnerAI?

FeltnerAI is a private, self-hosted chat server you control. It ships a standalone Rust
server, a shared React web app, and a cross-platform desktop **Portal** for connecting to one
or more servers. You bring your own [OpenAI-compatible provider](/docs/api/openai-compatible/),
and your accounts, chats, and provider credentials stay on your own machine.
